# MySprout（マイスプラウト）-気分記録アプリ-

## サービス概要
うつ病や双極性障害の方がその日の気分を記録するアプリ。<br>
記録を続けられるよう、記録のハードルを低くくなるようにしております。<br>
また、生活や治療に役立てられる機能もあります。(記録のグラフ化、医師に伝えたいことのメモ機能)<br>
　※メモ機能はMVPリリース後のアップデートで実装予定

## このサービスへの思い・作りたい理由
双極性障害になったことが、同じような方の手助けになるアプリを作成したいと思ったきっかけです。<br>

このアプリでは、以下のことを目的としています。
- 医師とのコミュニケーションをより円滑にし、適切な治療につなげる
- 記録のハードルを下げ、記録が続かないことによる自己嫌悪を防ぐ<br>

なぜこの2つを目的としているかというと、自身の体験からきています。<br>
特に以下の2点で、困ったので同じ悩みを抱えている方の助けになればと思っております。<br>
- 記録が続かないので、医師に正確な自分の状態を伝えられない<br>
- 記録が続かない自分への自己嫌悪<br>

最後ですが、ユーザー様からの声を反映して、たくさんの方が記録を続けられるアプリにしていければと思っております。

## ユーザーについて

うつ病の方や双極性障害の方<br>
特に、記録する気力がわかない方や記録が面倒だと感じる方

## 使い方のイメージ
### 記録
- 初めに<br>
記録方法は、5段階評価のアイコンをその日の気分で選択する方式です。<br>
以降は、アイコンと呼称させていただきます。
- アプリからの記録(当日の記録)
    1. アプリを開き、ホーム画面を表示
    2. ホーム画面に表示されているアイコンのいずれかをタップ
    3. GOODが出たら記録完了
- アプリからの記録(当日以外)
    1. アプリを開く
    2. ホーム画面の+ボタンを選択
    3. 記録画面が開くので今日の気分を選択
    4. 保存を押すと記録が完了
- ウィジェットからの記録(MVPでは未実装)
    1. ウィジェットからアイコンのいずれかをタップ
    2. GOODが表示され、気分の記録が完了

### 生活や治療への活用
- これまでの記録を曲線でグラフとして表示し、生活や治療に活用可能

## サービスの推しポイント
### 他サービスとの比較

| 評価項目 | 類似サービスA | 類似サービスB | 本サービス |
| ---------- | ---------- | ---------- | ---------- |
| 記録方法 | 5段階評価 | 11段階評価 | 5段階評価 |
| 記録ステップ数 | 3ステップ | 4ステップ | 2ステップ |
| 記録項目数 | 多数 | 2項目 | 1項目(最終的には3項目) |
| 特徴 | 評価アイコンをカスタマイズ可能 | 細かい気分の記録が可能 | 記録のハードルが低い |

※記入ステップ数は、気分のみを記録した場合。<br>
　記録する項目数が増えればそれに合わせて増えていきます。
### 推しポイント
1. 記録の手間の削減<br>
記録ステップ数を減らすことで記録の手間を削減。
2. 記録がシンプル<br>
記録項目が少ない＝情報量が少ないので重いうつ状態でも負担になりにくい。<br>
5段階評価は最高・良い・ふつう・悪い・最悪なので直感的に記録可能。
3. メモ機能(アップデート後)<br>
気分の記録と通院時の相談メモが1つのアプリで可能。<br>
類似サービスにはメモ機能がなく、別のアプリでメモの必要がある。

## 機能一覧
### MVPで作る機能
1. ユーザー登録/ログイン機能
2. 気分の記録機能
    - 5段階評価で記録
    - 記録したらGOODが表示
3. 気分のグラフ化
    - 1週間、1カ月単位でグラフを表示

### 実装予定の機能
- 1回目のアップデート<br>
    記録項目の拡充：エネルギー(やる気)を記録できるように<br>
    記録項目の拡充：睡眠時間を記録できるように<br>
    拡充項目のグラフ化：エネルギーと睡眠時間をグラフ化
- 2回目のアップデート<br>
    グラフ機能拡充：半年、1年単位でも表示できるように<br>
    メモ機能：通院時に医師に伝えたいことや言われたことをメモ可能
- 3回目のアップデート<br>
    ウィジェットから記録：記録項目をウィジェットから記録できるように<br>
                        ※Andoroid版を配信予定<br>
                        　iOS版は可能であれば配信
- 4回目のアップデート<br>
    記録日数のカウント：ホーム画面に合計記録日数を表示<br>
    育成要素：記録日数に応じて、ホーム画面の木が成長していく

## このアプリの懸念点とその対策
### 懸念点①
- 懸念点：技術的にウィジェットからの記録が可能なのか
- 理由：できるのならば他のアプリで類似サービスで行っていそうだから。
- 対策：通知から記録ができないかを模索する。
### 懸念点②
- 懸念点：ターゲットに使ってもらえるかが不安<br>
　　　　(リリースして改善の流れに持っていくことができるのか不安)
- 理由1：何の実績もないアプリを使ってもらえるのか
- 対策：自身のかかりつけの医師や医師のコミニティで意見を貰い改善に入っていく。<br>
　　　(専門的知見を貰うこちらの方が見てもらえるかわからないので難易度的に難しいか)
- 理由2：自分としては記録項目が多いと疲れてしまうが、実際は記録項目が多い方がいいのか。
- 対策：誰かに使用してもらってそういった声が多かったら、記録項目を追加していく。<br>
　　　または、別アプリとして記録項目追加版を作成する。<br>
　　　上記は時間があった場合の対処法で、時間がない場合の対処法は考えつかなかった。
- 理由3：実装する機能で躁状態やうつ状態の方が記録をできるようになるのか。<br>
　　　　かなり重いうつ状態は何もやる気が起きないので使用できるのか。
- 対策：使用しない方の理由を調査してブラッシュアップしていくしかないと考える。


## 発展の方向性
1. 記録が簡単をコンセプトとしつつ治療に役立つ機能や記録項目を増やす
2. 周りの人物がアプリを入れて患者の気分の状態を見れるようにする。<br>
これは自分としては、自分の状態を家族に見られるのが嫌な時期もあったので導入は慎重に考えたい
